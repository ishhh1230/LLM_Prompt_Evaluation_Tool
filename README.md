🧠 Prompt Evaluation & LLM Comparison Tool

A professional AI Trainer portfolio project that simulates real workflows used in LLM evaluation, prompt review, and response comparison.

This application helps reviewers analyze prompt quality, compare AI-generated responses, and maintain structured evaluation records.

Built as a Streamlit web application and deployable on Streamlit Cloud.

If deployed:

(https://llm-evaluation-tool.streamlit.app/)

📌 Project Overview

This tool simulates real tasks performed in roles such as:

1. AI Trainer

2. Data Annotator

3. LLM Evaluator

4. Prompt Engineer

4. AI Quality Analyst

The system provides a structured environment to:

1. evaluate prompt quality

2. compare multiple AI responses

3. score outputs using predefined evaluation metrics

4. track evaluation history

5. perform batch prompt review using CSV files

⚙️ Features

1️⃣ Prompt Quality Evaluation

Evaluate prompts based on multiple criteria:

1. Clarity

2. Specificity

3. Completeness

4. Ambiguity Control

5. The system generates:

6. total score

7. quality classification

8. improvement suggestions

2️⃣ LLM Response Comparison

Compare two AI-generated responses using structured metrics:

Evaluation metrics include:

1. Relevance

2. Clarity

3. Safety

4. Factuality

5. Instruction Following

The system determines:

1. total score for each response

2. the winning response

3. evaluation summary

3️⃣ Evaluation History

Store all review results in session history.

Features:

1. evaluation logs

2. reviewer name tracking

3. downloadable CSV export

4️⃣ Batch Prompt Review

Upload a dataset of prompts and evaluate them automatically.

Features:

1. CSV upload

2. prompt column selection

3. automated scoring

4. downloadable results

🧠 Example Use Case

Prompt:

1. Explain machine learning in simple terms for a beginner with one example.

Response A and Response B are evaluated using structured scoring criteria to determine which answer better satisfies the prompt.

📊 Evaluation Metrics

The system evaluates responses using the following scoring logic:

1. Relevance - 	Alignment with the prompt
2. Clarity - 	Readability and structure
3. Safety -	Absence of harmful content
4. Factuality -	Informational completeness
5. Instruction Following -	Compliance with prompt constraints
